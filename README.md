# raylib-template

A template library for building Raylib from source for C/C++

#### Running
Can be ran as any other CMake project  
`cmake -B [build folder] -S .`  
`cmake -build [build folder]`  
`./[build folder]/RaylibTemplate `  

#### Makefile
A helper Makefile exists to simplify the process above, with the commands  
`make debug_build`  
`make debug_run`  
`make release_build`  
`make release_run`   
`make clean`
